Abstract

Dexterous grasping has advanced rapidly in recent years, yet bimanual manipulation for large‑scale objects remains underexplored due to the increased complexity of coordinating two hands while considering object geometry, environmental constraints, and potential hand‑hand conflicts. In this work, we propose an efficient method for generating high‑quality bimanual grasp datasets, which outperforms prior approaches in both generation speed and success rate. Using this pipeline, we generate and physically validate 5.3 million successful bimanual grasp pairs. Furthermore, we develop a diffusion‑based model trained on this synthetic dataset to infer feasible bimanual grasp poses from partial object point clouds. This model achieves a 80.7\% success rate in simulator based on partial point cloud and 46.3\% in real experiments.
